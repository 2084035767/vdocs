# 七、调试技巧

## 7.1 错误类型和 Debugging 工具

#### 断点调试

1. 浏览器中按 F12–> sources -->找到需要调试的文件–>在程序的某一行设置断点(在行数点一下)
2. 刷新浏览器
3. Watch: 监视，通过 watch 可以监视变量的值的变化，非常的常用
4. F11: 程序单步执行，让程序一行一行的执行，这个时候，观察 watch 中变量的值的变化
