## 3.3 栈 (Stack)

「栈 stack」是一种遵循先入后出的逻辑的线性数据结构。



我们把堆叠元素的顶部称为“栈顶”，底部称为“栈底”。将把元素添加到栈顶的操作叫做“入栈”，删除栈顶元素的操作叫做“出栈”。



栈的常用操作如表 5-1 所示，具体的方法名需要根据所使用的编程语言来确定。在此，我们以常见的 `push()`、`pop()`、`peek()` 命名为例。

栈的操作效率

| 方法   | 描述                   | 时间复杂度 |
| :----- | :--------------------- | :--------- |
| push() | 元素入栈（添加至栈顶） | \(O(1)\)   |
| pop()  | 栈顶元素出栈           | \(O(1)\)   |
| peek() | 访问栈顶元素           | \(O(1)\)   |
